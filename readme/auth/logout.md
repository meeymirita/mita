====================
ВЫХОД ИЗ СИСТЕМЫ (LOGOUT)
====================

Метод: POST
Эндпоинт: /api/logout

Требования:
- Требуется Bearer Token в заголовке
- Тело запроса пустое (не требуется отправлять данные)

Заголовок запроса:
Authorization: Bearer eyJ0eXAiOiJKV1QiLCJhbGciOiJIUzI1NiJ9...

Процесс:
1. Сервер получает запрос с Bearer Token
2. Проверяет валидность токена
3. Удаляет ВСЕ токены пользователя из базы данных
4. Возвращает подтверждение выхода

Успешный ответ:
{
"status": "Вы вышли из аккаунта"
}
